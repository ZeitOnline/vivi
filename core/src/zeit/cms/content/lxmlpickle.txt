lxml pickle support
-------------------


Verify the lxml pickle support.

>>> import pickle
>>> import lxml.etree
>>> import zeit.cms.content.lxmlpickle
>>> xml = lxml.etree.fromstring('<foo><b>zoot</b></foo><?bar?>')
>>> p = pickle.dumps(xml)
>>> restored_xml = pickle.loads(p)
>>> print(zeit.cms.testing.xmltotext(restored_xml.getroottree()))
<foo>
  <b>zoot</b>
</foo>
<?bar?>
