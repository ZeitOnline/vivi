XML block
=========

The xml block contains raw xml[#functional]_.

>>> import zeit.content.cp.centerpage
>>> cp = zeit.content.cp.centerpage.CenterPage()
>>> lead = cp['lead']
>>> import zeit.content.cp.interfaces
>>> import zope.component
>>> factory = zope.component.getAdapter(
...     lead, zeit.content.cp.interfaces.IBlockFactory, name='xmlblock')
>>> xmlblock = factory()

An empty XML block contains an empty <container> element:

>>> import lxml.etree
>>> container = xmlblock.xml
>>> container
<Element container at ...>
>>> print lxml.etree.tostring(container, pretty_print=True)
<container.../>

It is of the type 'xmlblock':

>>> container.get('{http://namespaces.zeit.de/CMS/cp}type')
'xmlblock'

Also, it has a name:

>>> container.get('{http://namespaces.zeit.de/CMS/cp}__name__') != None
True

You can add elements to the container:

>>> container.append(lxml.etree.Element('foo'))
>>> print lxml.etree.tostring(container, pretty_print=True)
<container ...>
  <foo/>
</container>
