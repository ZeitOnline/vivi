0.23.1 (unreleased)
-------------------

- Moved reusable styles to zeit.cms.
- Add 'type' field to search result (for #9390).
- Lowercase autocomplete query and use right truncation to improve search
  results.
- Provide JS-API to filter searches for types (for #10698).

-------------------
