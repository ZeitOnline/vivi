==========
References
==========

Feed
====

Feeds are references via xi:include. Create a feed mock:

>>> class Feed(object):
...     pass

Create a feed and adapt it to the reference:

>>> from zeit.xmleditor.reference import FeedXMLReference
>>> feed = Feed()
>>> feed.uniqueId = 'feed-unique-id'
>>> xml = FeedXMLReference(feed)
>>> xml
<Element {http://www.w3.org/2001/XInclude}include at ...>

The whole structure also has a fallback:
>>> import lxml.etree
>>> print lxml.etree.tostring(xml, pretty_print=True)
<xi:include xmlns:xi="http://www.w3.org/2001/XInclude"
    href="feed-unique-id">
  <xi:fallback>Der Feed ist im Moment leider nicht erreichbar.</xi:fallback>
</xi:include>
